## Como instalar a aplicação

Basta apenas baixar o projeto, e seguir os seguintes passos:

- Abrir a pasta `client` no terminal e digitar o comando `npm install` ou `yarn install`, que ele ira instalar todas as bibliotecas. Após instalar as dependencias basta apenas executar o comando `npm run dev` ou `yarn dev` para rodar o frontend.

- Abrir a pasta `server` no terminal e digitar o comando `npm install` ou `yarn install`, que ele ira instalar todas as bibliotecas. Após instalar as dependencias basta apenas executar o comando `npm start` ou `yarn start` para rodar o backend.

## Como rodar a aplicação localmente

#### Frontend

Para startar o frontend localmente, você pode digitar o comando `npm run dev` ou `yarn dev` no diretório do frontend.

Após isso ele irá abrir na porta `8080` no seu localhost, ou seja, basta apenas entrar em `localhost:8080`.

#### Backend

Para startar o backend localmente, você pode digitar o comando `npm start` ou `yarn start` no diretório do backend.

Após isso ele irá abrir na porta `3001` no seu localhost.
